IP Forger

IP Forger is an app with the goal of crafting and sending networking packets and at the same time, helping the user better understand the fields in each header. IP Forger as also a feature to scan the network for active IP's by sending an ICMP Scan.

Features

Sending Packets

With IP Forger you can send your own customized TCP,UDP and ICMP packets!

Scanning IP's

You can discover all active IP's in your network by doing a classic IP scan and get a report with the result.

Learning

Do you find all this overwhelming? Don't worry, everything is documented, so you learn how everything works!
